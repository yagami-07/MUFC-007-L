# Citation Modes

Use this reference only when detailed citation handling is needed.

## Required Source Policy

- Prefer sources published within 5 years of the writing date.
- Require DOI for ordinary journal articles unless the item is a justified
  exception.
- Do not invent DOI, pages, volume, issue, publisher, author names, dates, or
  journal titles.
- Verify uncertain bibliographic data from current sources before final use.

## GB/T 7714-2025

Default for CSSCI, AMI, Chinese dissertations, and Chinese humanities/social
science manuscripts.

Article:

```text
[序号] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码. DOI: DOI号.
```

Book:

```text
[序号] 作者. 书名[M]. 出版地: 出版者, 年.
```

Chapter:

```text
[序号] 作者. 析出文献题名[M]//编者. 书名. 出版地: 出版者, 年: 起止页码.
```

## APA

Default for many SSCI social science manuscripts.

Article:

```text
Author, A. A., & Author, B. B. (Year). Article title. Journal Title, volume(issue), page range. https://doi.org/DOI
```

Book:

```text
Author, A. A. (Year). Book title. Publisher.
```

## MLA

Use for MLA-requested humanities manuscripts.

Article:

```text
Author Last Name, First Name. "Article Title." Journal Title, vol. volume, no. issue, Year, pp. page range. DOI.
```

Book:

```text
Author Last Name, First Name. Book Title. Publisher, Year.
```
